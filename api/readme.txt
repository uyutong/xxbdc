1. 资源访问路径
upload
    ├─book....................书的封面图片地址
    ├─exercise
    │  ├─img ................ 练习中的图片地址
    │  └─mp3 .................练习中的音频地址
    ├─unit ....................单元图片
    └─	
        ├─example .............单词例句/搭配音频地址
        ├─img .................单词图片
        ├─mp3..................单词音频
        └─mp4..................单词视频

2. 题型硬编码（type 属性）
type = 1 ..................看图选择题
type = 2 ..................选图填空
type = 3 ..................句子排序
type = 4 ..................听力单项选择
type = 5 ..................听力填空匹配
